# Netflix Data Analysis

## About

This project analyzes the Netflix Movies and TV Shows dataset using Python and Pandas.

The goal is to explore Netflix content and answer several business questions through data analysis and visualization.

## Technologies

* Python
* Pandas

## Questions

The analysis includes:

1. How has the amount of content changed over the years?
2. Which countries produce the most content?
3. Are movies or TV shows more common?
4. What are the most popular genres?
5. Which ratings appear most often?

## Dataset

Dataset: Netflix Movies and TV Shows

Main file:

```text
netflix_titles.csv
```


## Results

The project provides:

* Data cleaning
* Exploratory Data Analysis (EDA)
* Data visualizations
* Business insights
